# Inaugurator

## List of available commands

1. create
2. generate
3. install
4. list

## Naming Convention

1. Files and folder names will begin with __pc__name
2. To put template in code use {{ToPascalCase .}}

Allowed Case:  

* pc : PascalCase
* cc : camelCase
* ss : snake_case
* lkc : lowercase-kebab-case
