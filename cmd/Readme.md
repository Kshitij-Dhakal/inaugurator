# Command examples

1. Create Boilerplate  
    `inaugurator create ng -f config.json`

2. Test Boilerplate
   * `inaugurator generate -f UserComponent.html`
   * `ng generate component User`

3. Install Boilderplate  
    `inaugurator install ng`

4. List Boilderplate
    * `inaugurator list`
    * `ng list`
